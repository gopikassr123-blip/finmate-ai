# FinMate AI – Intelligent Financial Assistance Chatbot

FinMate AI is an AI-powered financial assistance chatbot designed to provide users with personalized financial guidance through conversational interaction. The application focuses on budgeting support, expense analysis, money-saving strategies, and financial planning for students and beginners.

The chatbot is developed using Flask for backend processing and integrates with the Gemini API to generate intelligent responses in real time. The project demonstrates the practical implementation of Generative AI concepts within a full-stack web application.

---

## Features

- AI-powered conversational finance assistant
- Personalized budgeting suggestions
- Expense analysis and spending insights
- Financial planning guidance
- Student-focused money management support
- Real-time chatbot interaction
- Responsive web interface
